# Core Person Changelog

## Introduction

This document describes the (major) changes to [the current version 2.1.1](https://github.com/SEMICeu/Core-Person-Vocabulary/tree/master/releases/2.1.1) of the Core Person Vocabulary for which a new version is being proposed ([version 2.1.2](https://semiceu.github.io/Core-Person-Vocabulary/releases/2.1.2/)). The list of changes results in the new version to be considered as a bug fix release.

All of the changes were discussed and approved during the following webinars:

    <li><a href="https://interoperable-europe.ec.europa.eu/collection/semic-support-centre/event/webinar-core-vocabularies">Webinar</a> of 25/02/2025</li>

</ul>
